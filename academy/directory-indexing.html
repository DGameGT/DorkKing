Menguasai Directory Indexing ("Index of")

Diupdate: 11 Januari 2026 ‚Ä¢ Kategori: Reconnaissance ‚Ä¢ Level: Pemula

Directory Listing (atau Directory Indexing) adalah fitur standar pada server web (Apache, Nginx, IIS) yang menampilkan daftar file di dalam folder jika tidak ada halaman default (seperti index.html atau index.php). Jika fitur ini aktif pada folder yang sensitif, ini adalah bencana keamanan.

Bagi pemburu OSINT, ini adalah harta karun.

1. Anatomi "Index of"

Ketika directory listing aktif, halaman web biasanya memiliki judul (Title) yang sangat khas, yaitu Index of /namafolder. Google mengindeks judul ini.

Query dasar untuk menemukannya:

intitle:"index of"

Namun, itu terlalu umum. Untuk hasil yang efektif, kita perlu mengombinasikannya dengan:

  • Keywords spesifik (apa yang kita cari).
  • Format file (ekstensi).
  • Struktur direktori umum (Parent Directory).

2. Teknik Mencari File Media (MP4/MP3/Ebook)

Salah satu penggunaan paling populer dari dork ini adalah untuk mencari film, lagu, atau buku tanpa masuk ke situs streaming yang penuh iklan.

Mencari Film/Video:

intitle:"index of" (mp4|mkv|avi) "Avengers" -html -htm -jsp -php
Penjelasan:
  • (mp4|mkv|avi): Mencari salah satu ekstensi video ini.
  • -html -htm: Menghilangkan hasil halaman web biasa, fokus pada direktori server.
  • "Avengers": Kata kunci film.

Mencari Ebook (PDF/EPUB):

intitle:"index of" (pdf|epub|mobi) "Hacking" "Tutorial"

3. Teknik Mencari Data Sensitif

Di dunia keamanan siber, kita menggunakan teknik yang sama untuk menemukan data kebocoran (Data Leaks).

Mencari File Backup Admin:

intitle:"index of" "backup" OR "bak" OR "old" site:go.id

Query ini akan mencari folder bernama backup di domain pemerintah Indonesia.

Mencari Database (DCIM/Camera Uploads):

Seringkali orang melakukan backup foto HP ke server web tanpa proteksi.

intitle:"index of" "DCIM" OR "Camera"

Mencari Log Chat WhatsApp:

intitle:"index of" "WhatsApp Images" OR "WhatsApp Databases"

4. Server Spesifik (Apache / Nginx / IIS)

Setiap server memiliki "signature" teks yang berbeda di halaman index-nya.

  • Apache: intitle:"Index of" "Apache Server at"
  • Nginx: intitle:"Index of" "Welcome to nginx" (jarang, karena nginx defaultnya mematikan autoindex)

5. Cara Mencegah (Untuk Admin)

Jika Anda adalah admin server, matikan fitur ini segera.

  • Apache (.htaccess): Tambahkan baris Options -Indexes
  • Nginx (nginx.conf): Pastikan autoindex off; (ini default, tapi periksa lagi).
